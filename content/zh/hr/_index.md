# 招聘 OpenAnolis开源社区运营高级专家

阿里云智能操作系统团队负责阿里云的服务器操作系统的研发和产品化工作，目前已经形成 Alibaba Cloud Linux 、安全容器等多个产品和系统支撑阿里业务的发展。随着新硬件和新应用的发展，尤其是云基础设施的发展，对系统软件提出全新的要求，同时也是系统软件创新发展的新机遇。如果你想在服务器操作系统、容器技术、基础软件领域不断挑战和攀登，欢迎加入我们。你不仅可以跟传说中的多隆大神一起工作，还可以参与到云计算的发展大潮中。阿里巴巴操作系统研发团队期待你的加入！

### 岗位职责：

1. 负责OpenAnolis操作系统开源社区运营，提升OpenAnolis在业界的影响力和构建开发者生态；
2. 制定开源技术、社区整体运营规划，为社区发展及各项指标负责；
3. 负责协助开源技术生态运营，推动与IHV，OSv，ISV等生态合作伙伴的合作；
4. 跨部门协调项目所需的合作团队，落实项目的阶段性结果并实现最终目标；

### 岗位要求：

1. 对开源理念及相关技术有深入的理解，有实际的社区活动策划经验，在开源相关领域有一定的行业影响力；
2. 在操作系统，云原生技术等领域有技术布道，解决方案架构，市场拓展等方面的经验；
3. 有良好的人际交往技巧，以及文字写作能力，英语水平满足工作交流；
4. 具备良好的学习能力，心态积极进取，良好的沟通协调能力，拥有较强抗压能力；
